# 시그널

Temporary no-login party web app prototype.

This static MVP demonstrates:

- People list
- Anonymous Signal flow
- Match and notification screens
- Play section
- Topic Roulette
- Dice Mission
- Ladder Game
- Random Pairing
- AI Icebreaker placeholder

This version is designed for GitHub Pages preview. Supabase, OpenAI server routes, realtime notifications, and host admin tools should be connected in the production Next.js app.
